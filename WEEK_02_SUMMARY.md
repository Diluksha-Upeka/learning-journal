## Day 13 — JSON (File Storage / Persistence)

- **Serialization** means: Python dict/list → JSON text → saved to a file.
- **Deserialization** means: JSON text from file → Python dict/list.

### Functions to memorize
- `json.dump(data, file)` writes Python → JSON (save)
- `json.load(file)` reads JSON → Python (load)
